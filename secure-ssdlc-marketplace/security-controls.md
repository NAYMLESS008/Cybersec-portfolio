# Security Controls — Multi-Vendor Marketplace

## Controls

| Control | Purpose |
|---|---|
| MFA | Reduces account takeover risk |
| Login Lockout | Limits brute-force attempts |
| CAPTCHA | Reduces automated abuse |
| Activity Logging | Supports accountability and investigation |
| Backups | Supports recovery from failure or compromise |
| Security Headers | Reduces browser-based attack risk |
| Password Policy | Improves account security |
| Role-Based Access Control | Limits privilege misuse |
| Restricted Purchasing | Prevents anonymous checkout abuse |
