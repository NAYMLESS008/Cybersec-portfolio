# Secure SSDLC for Multi-Vendor Marketplace

## Project Overview

This project involved designing and documenting a secure software development lifecycle for an Etsy-like WordPress/Dokan multi-vendor marketplace.

The project focused on requirements, threat modelling, vulnerability analysis, security controls, testing evidence, and risk assessment.

## Tools and Frameworks

- WordPress
- Dokan
- Burp Suite
- STRIDE
- OWASP ASVS
- MITRE ATT&CK
- MFA
- CAPTCHA
- Login lockout
- Activity logging
- Backups
- Security headers

## Security Controls Implemented / Documented

- Multi-factor authentication
- Login lockout
- CAPTCHA
- Activity logging
- Backups
- Security headers
- Password policy
- Role-based access control
- Restricted purchasing for logged-in users
- Stripe test mode
