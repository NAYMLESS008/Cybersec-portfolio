## Featured Projects

### 1. Multi-Region T-Pot Honeypot Threat Monitoring

A cloud-based honeypot monitoring project using T-Pot, Suricata, ELK/Kibana, Docker, Cowrie, Dionaea, Honeytrap, and related honeypot services.

Key areas covered:

- Real-world attack telemetry collection
- Suricata IDS alert analysis
- ELK/Kibana log investigation
- SSH brute-force activity review
- Internet scanning behaviour
- SOC-style findings and reporting

### 2. SOC Alert Investigation

A practical SOC-style investigation report covering alert review, evidence collection, MITRE ATT&CK mapping, verdict classification, and recommended response actions.

Key areas covered:

- Alert triage
- Log correlation
- Suspicious activity analysis
- True positive / false positive classification
- Incident response recommendations

### 3. Secure SSDLC for Multi-Vendor Marketplace

A secure software development lifecycle project for a WordPress/Dokan multi-vendor marketplace.

Key areas covered:

- STRIDE threat modelling
- OWASP ASVS control mapping
- MITRE ATT&CK mapping
- Burp Suite security testing
- MFA, login lockout, CAPTCHA, activity logging, backups, and security headers

### 4. Malware Analysis Notes

A study-based malware analysis section covering static analysis, dynamic analysis, malicious documents, indicators of compromise, MITRE ATT&CK mapping, and malware report writing.
